# Homework: Handling events

Create a JavaScript program that transforms `<section>` elements inside
a `<div class="tabs">` into a tabbed interface. A tabbed interface is a
common UI pattern where content is divided into multiple panels, but
only one is visible at a time. Each panel is accessed by clicking a tab,
similar to how tabs work in a web browser. If a section has a
`tab-title` attribute, use its value as the tab's title; otherwise, use
the section's position (1, 2, 3, ...) as the title.
