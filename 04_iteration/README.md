# Homework: Iteration

For this assignment, your task is to implement the following functions:

- A function named `toUpper` that transforms a given array of strings
  into uppercase.

- A function named `vowelCount` that returns the number of vowels in a
  given word.

- A function named `isSuperset` that determines if the numbers in a
  given array are included in a second given array of numbers.

- A function named `avg` that returns the average of a given array of
  numbers.

- A function named `orderedList` that transform a given array of strings
  into an HTML ordered list.

- A function named `reverse` that reverses the characters in a given
  string.
